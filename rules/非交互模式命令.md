所有命令必须使用 **非交互模式（自动 yes/默认值）**，
禁止任何需要手动确认、输入密码、等待回车的交互式命令。

约束规则：
- apt → apt -y
- yum → yum -y
- npm → npm install -y
- git 相关：
  - 禁止任何交互式 Git 命令（git push、git pull、git add -i 等）
  - git show 必须非交互、不打开分页器，写法固定为：git show --no-pager
  - git 必须加 --no-interactive 或 -y
- rm → 必须用 rm -f，禁止 rm -i
- 任何需要 [Y/n] 的命令都要加自动确认参数
- Python 代码禁止用 input()，改用环境变量/文件传参
- 禁止任何需要等待用户输入的操作
