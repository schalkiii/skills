# Skills

AI 编程助手专用技能（Skills）集合，提供领域专业知识与工作流指导。

## 技能列表

### verilog-design

工业级 Verilog / SystemVerilog RTL 设计方法学，覆盖数字 IC 设计的完整生命周期。

**核心能力：**

- **规格定义** — 接口契约、流水线架构模板、参数化设计
- **RTL 编码** — 组合/时序分离、防锁存器、总线规范、有符号运算
- **数值系统** — 位宽审计、溢出保护、补码运算、浮点流水线
- **测试平台** — 自检测试平台、验证策略
- **调试** — 四阶段渐进调试法：定位 → 广角快照 → 聚焦追踪 → 迭代深入
- **综合** — 时序收敛、面积优化、功耗分析
- **AI 波形调试** — 开源 EDA 工具链与自动化工作流

**触发条件：** 编写或审查 Verilog / SystemVerilog RTL 代码、设计数字电路流水线、规划模块接口与数值系统、创建自检测试平台、调试仿真波形、进行逻辑综合或低功耗优化。

**设计原则：** 所有问题从门电路、触发器、组合/时序逻辑、时钟域、位宽、总线等硬件底层视角分析，禁止软件编程思维。
